Reflectivity

        reflect reflectivity
Controls the surface reflectivity. If the value is 0 (or it is omitted) then the surface will not be reflective; if the value is 1 a mirror-like reflectivity will result. The specular keyword controls the colour of the reflected light.

In this example a mirror is shown with two spheres visible in the mirror, the green sphere should be directly visible but the red sphere is (almost) hidden from direct view behind the green sphere!

Figure 4-3

        /*
          mirror.ray
          A mirror with 2 spheres
          sp  8 mar 92
        */
        eyep  7 7 5
        lookp 0 1 3.5
        screen 900 600
        background .9 .9 .9

        /* mirror */
        box  specular 1 1 1 reflect 1
            .1 1 2  0 3 5

        /* two spheres */
        sphere ambient .5 0 0
             .5   1.5 .5 2.5

        sphere ambient 0 .5 0
             .5   2.5 1.5 3

        /* wall behind mirror */
        box ambient 0 0 .5
             -0.1 0 1  0 4 6
